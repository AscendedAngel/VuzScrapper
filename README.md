# Vuz Scrapper

**Парсер и симулятор конкурсных списков**  
Программа анализирует открытые конкурсные списки бакалавриата, учитывая приоритеты абитуриентов, и вычисляет **реалистичное текущее место** с исключением ***"мёртвых душ"*** - тех, кто гарантированно поступает на более приоритетную программу.

> 🎯 **Цель** - помочь абитуриенту понять свои реальные шансы, пока официальные списки ещё полны хаоса (многократные заявки одних и тех же людей на разные направления).

## 📌 Возможности

- Парсинг нескольких конкурсных списков (направлений) с сайта [abit.itmo.ru](https://abit.itmo.ru), а также [enrollelists.spbu.ru](https://enrollelists.spbu.ru/reports/PriemList02.php?mode=filter&education_level_sort_order=1).
- Учёт **всех квот**:
    - олимпиадники (БВИ)
    - целевая квота (ЦК)
    - особая квота (ОсК)
    - отдельная квота (ОтК)
    - общий конкурс (ЕГЭ)
- Учёт **приоритетов программ** у каждого абитуриента.
- **Симуляция зачисления**:
  - Абитуриент, который гарантированно проходит на свою самую приоритетную программу (по баллам и местам), ***"зачисляется"*** туда.
  - После этого он **удаляется из остальных списков**, освобождая места для других.
- Вывод вашего актуального места (с учётом симуляции) по каждому направлению.

## ⚙️ Как работает алгоритм

1. **Сбор данных** - парсятся страницы рейтингов для заданных направлений.
2. **Кластеризация** - абитуриенты группируются по типу конкурса (олимпиадники, целевики, особые, отдельные, общий).
3. **Сортировка** - внутри каждой группы кандидаты сортируются по убыванию суммы баллов (ЕГЭ + индивидуальные достижения).
4. **Распределение мест** - для каждого абитуриента проверяются его заявки в порядке приоритета:
   - Если на первом (самом приоритетном) направлении в его квоте есть свободное место - он зачисляется туда, и все остальные его заявки удаляются.
   - Если места нет - проверяется следующее по приоритету направление.
   - Если нигде нет мест - он остаётся во всех списках (как ***"не прошедший"*** в симуляции).
5. **Результат** - выводится обновлённое место для каждого направления, где вы присутствуете.
